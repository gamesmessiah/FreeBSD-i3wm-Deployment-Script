# FreeBSD-i3wm-Deployment-Script
<p>This script will download and setup i3wm and dmenu for freebsd.</p>

<p>It will configure rc.conf and install software.</p>

<p>xorg<br>

lxappearance<br>
compton and compton-conf<br>
Arandr
